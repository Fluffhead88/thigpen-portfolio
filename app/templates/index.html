My Record Bin

My final project for Carolina Code School graduation. This is a web app that allows users to catalog and organize their vinyl record collection. It fetches information from Last.FM's API and saves album information to a users personal collection. You can also view other users collections and contact them from the app which uses mailgun to send an email allowing them to purchase or trade albums with others. Back end is build using Python/Django and front end is built using JavaScript/React with Material UI.
